Serverless payroll-data ingestion platform on AWS (Lambda, SQS, DynamoDB, CDK), built as a CMU MSE Studio project for [Troutwood](https://www.troutwood.com/). Designed the end-to-end CI/CD platform on GitHub Actions: 14 of 17 workflows, gated deploys, per-branch ephemeral stacks, GitHub OIDC auth, and LocalStack integration tests.
